# 分子与光 — Trae SOLO 技能创作赛参赛作品

## 一、Skill 简介

**「分子与光」**是一款面向大学物理教学的虚拟仿真实验 Skill，基于 Trae SOLO 开发，在 AetherViz Master 开源框架基础上实现了光子与分子共振吸收/发射的完整物理闭环可视化。

> **致谢**：本项目基于开源框架 [AetherViz Master](https://github.com/andyhuo520/aetherviz-master) 开发，感谢原作者 Andy Huo 的贡献。本项目在框架基础上独立实现了物理仿真逻辑、分子建模、UI 交互等核心功能。

学生无需实验室设备，打开浏览器即可：
- 选择 8 种真实分子（CO/N₂/O₂/CO₂/CH₄/H₂O/NO₂/O）
- 调节入射光频率（0~9.0×10¹ Hz）
- 实时观察光子被分子吸收、分子振动、再发射光子的全过程
- 对比共振吸收与光子穿透的不同现象

**技术亮点**：Three.js 3D 渲染 + 阻尼谐振物理模型 + 频率-颜色实时映射 + 科学计数法专业显示

## 二、使用场景

### 场景 1：大学物理课堂辅助教学
教师在讲解"分子振动光谱"章节时，直接打开仿真页面，现场演示不同频率入射光对分子的影响，比静态图片更直观。

### 场景 2：学生自主探究学习
学生课后自主调整参数，探索"为什么水分子固有频率远高于氮气"、"改变入射光频率会发生什么"等问题，培养科学探究能力。

### 场景 3：实验教学替代方案
在缺乏实验设备的学校，虚拟仿真可作为实验教学的有效补充，确保每位学生都能观察到分子共振现象。

## 三、创作过程

### 1. 需求分析与物理建模
- 确定 8 种教学常用分子的固有频率数据（来源于标准光谱学数据）
- 设计光子共振吸收的判定逻辑：|f_incident - f_natural| ≤ Δf_threshold
- 采用阻尼谐振模型模拟分子振动：A(t) = A₀ · e^(-decay·t) · sin(f·t)

### 2. 技术选型
- **3D 引擎**：Three.js r134（轻量，CDN 加载，无需构建）
- **UI 框架**：TailwindCSS v3.4（快速原型）
- **公式渲染**：KaTeX 0.16.11（物理公式美观展示）
- **架构**：单文件 HTML（零依赖，双击即可运行）

### 3. 关键提示词示例

**分子数据定义提示词**：
```
每个分子定义包含：中文名、化学式、固有频率(THz)、波段、原子数组（元素符号/半径/颜色/3D坐标）、键合关系数组
```

**相机配置提示词**：
```
PerspectiveCamera(55, aspect, 0.1, 1000)
position: (5, 3, 6) 右上方视角，确保光源和分子同时可见
target: (0, 0, 0) 聚焦分子中心
dampingFactor: 0.08 平滑跟随
maxPolarAngle: π×0.85 防止观察到底面
```

**光子物理提示词**：
```
光子生命周期：traveling → absorbing → emitted
碰撞检测：距分子中心 < 0.6 时触发频率匹配
共振吸收：|Δf| ≤ f₀×2% 时吸收并触发振动
非共振：光子穿透分子继续传播
```

## 四、效果展示

### 核心功能截图说明

1. **主界面**：3D 场景中银色手电筒发射光子飞向水分子模型
2. **控制面板**：入射光频滑块（320px）+ 8 个光频选择按钮（78px 固定宽）
3. **共振吸收**：当入射光频率 = 分子固有频率时，分子开始剧烈振动并发射新光子
4. **光子穿透**：频率不匹配时，光子直接穿过分子继续传播
5. **电磁波谱**：点击"显示光谱"查看分子位置在电磁波谱中的标记

### 用 Trae SOLO 之前 VS 之后

| 对比项 | 用 Trae SOLO 之前 | 用 Trae SOLO 之后 |
|--------|------------------|------------------|
| 开发时间 | 需要搭建完整前端工程（Webpack/Vite + 多文件管理） | 单文件 HTML，直接编写，零配置 |
| 3D 实现 | 需要学习 Three.js 完整工作流 | 通过提示词描述 3D 场景，AI 自动生成代码 |
| 物理逻辑 | 手动编写碰撞检测、状态机等复杂逻辑 | 通过提示词描述物理规则，AI 自动实现 |
| UI 对齐 | CSS 调试耗时，反复修改 margin/padding | 通过提示词描述对齐要求，AI 精确调整 |
| 科学计数法 | 手动实现数字格式转换 | 提示词描述需求，AI 自动生成转换函数 |
| 文档产出 | 需额外编写文档 | Skill 文档、用户手册、Wiki 自动生成 |

## 六、创新点

1. **完整物理闭环**：不仅仅是视觉展示，实现了从光子发射到分子吸收再到光子发射的完整物理过程
