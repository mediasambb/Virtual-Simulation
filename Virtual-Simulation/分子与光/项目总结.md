### 物理闭环 ✅ 100%

```
选择分子 → 设置入射光频率 → 发射光子 → 碰撞检测
  → 共振(|Δf|≤阈值): 吸收 → 振动(2.8s阻尼) → 发射新光子 → IDLE
  → 非共振: 光子穿透 → 继续传播
```
